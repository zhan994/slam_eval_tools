# slam_eval_tools

***Python package for the evaluation of odometry and SLAM based on core code of  [evo 1.13.1](https://github.com/MichaelGrupp/evo/tree/86f52ade6da8cc4749c6170b1d2771ea1e0f1c66).***

## Install

To run the scripts in folder `tools/`, please install the official `evo` package to avoid missing dependencies.

```bash
pip3 install evo
```
